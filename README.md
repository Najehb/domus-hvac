# DOMUS HVAC - Sistema de Gestión de Tickets

Sistema para gestionar incidencias del sistema HVAC en la Torre DOMUS (13 niveles).

## 🏢 Estructura del Edificio
- **Lobby** (Nivel 1)
- **Niveles 2 al 13**

## 🛠️ Características
- Creación de tickets por nivel
- Dashboard con resumen del proyecto
- Sistema de logs de actividad
- Filtrado por estado y nivel
- Almacenamiento local (no requiere servidor)

## 🚀 Cómo usar
1. Abrir `index.html` en un navegador web
2. Usar el menú lateral para navegar
3. Crear tickets desde "Nuevo Ticket"
4. Monitorear desde el Dashboard

## 💾 Datos
Los datos se almacenan en el navegador (localStorage).
Para respaldar, exportar los datos desde las herramientas de desarrollador.
